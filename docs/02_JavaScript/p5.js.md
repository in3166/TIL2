### Circle()
`circle(x, y, d)`
- x: 원 중심정의 x좌표
- y: 원 중심정의 y좌표
- d: 원의 지름
...

### 그림 채우기와 태두리
- 도형을 채우기
  - 도형을 그리기 전에 선언 (선언 다음의 모든 도형 적용됨)
  - `fill(red, blue, green, [alpha])`, alpha: 0~255: 투명도

```javascript
function setUp(){
  createCanvas(400, 400);
  stroke('blue');
    strokeWeight(10);
  fll('red');
  rect(100, 100, 100);
}
```

- nofill(): 투명
- 태두리: `stroke('색');, strokeWeight(10);`
