Docsify와 Netlify를 사용해 웹 사이트로 배포했습니다.

<br>

- Doscify 사용

```md
npm i docsify-cli
docsify init ./docs
docsify serve docs
```

- 사이드바 자동 생성

```md
npm i docsify-tools -g
docsify-auto-sidebar -d docs
```
